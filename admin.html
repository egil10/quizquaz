Firebase Setup (One-Time)

Set up Firebase to publish quizzes instantly without GitHub commits! This only needs to be done once.

Steps to set up:

  1. Go to Firebase Console
  2. Create a new project (or use existing)
  3. Enable Firestore Database
  4. Go to Project Settings > Your apps > Web app
  5. Copy the config values
  6. Edit js/firebase-config.js with your values
  7. Refresh this page

Why Firebase? No GitHub commits needed! Quizzes publish instantly and work from anywhere. Free tier is generous for this use case.
